Qué es DNS

El Sistema de Nombres de Dominio (DNS, por sus siglas en inglés) le permite acceder a un sitio web a través de una dirección web amigable con el usuario como www.binary.com, en luga de una complicada dirección IP.

The default DNS services provided by your ISP can be slow and insecure. Binary.com recommends using Cloudflare's 1.1.1.1 DNS service that gives you a faster and more private way to browse the Internet.
