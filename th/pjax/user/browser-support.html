ปัญหาการเข้าสู่ระบบ

หากท่านไม่สามารถเข้าใช้งานบัญชีของท่าน โปรดตรวจสอบคุกกี้ (Cookies) ของเบราเซอร์ท่าน เว็บไซต์ของเราใช้คุกกี้ที่เก็บอยู่ในบัญชีผู้ใช้งานและรหัสผ่านของท่าน (ที่มีการเข้ารหัส) ในเครื่องคอมพิวเตอร์ของท่านในขณะที่ท่านใช้งานระบบซื้อขาย

เพื่อติดตั้งคุกกี้ โปรดปฏิบัติตามขั้นตอนข้างล่างสำหรับเวอร์ชันของเบราเซอร์ที่ท่านใช้งาน

Binary (C.R.) S.A.

Chrome 51+

  • ไปที่ "เมนู Chrome (ปรับแต่งและควบคุม Google Chrome)" และเลือกการตั้งค่า
  • คลิกที่แสดงการตั้งค่าขั้นสูง
  • คลิกที่ปุ่มการตั้งค่าเนื้อหาภายใต้เมนูความเป็นส่วนตัว
  • เลือกอนุญาตให้ตั้งค่าเฉพาะข้อมูลในเครื่อง (แนะนำ)
  • คลิกที่ปุ่มเสร็จสิ้นเพื่อบันทึกการตั้งค่าของท่าน
Binary (IOM) Ltd

Mozilla Firefox 47+

  • เปิดเมนู และเลือก การตั้งค่าที่ต้องการ
  • คลิกที่แท็บความเป็นส่วนตัว
  • ตั้ง "Firefox จะ" เป็น "จำค่าเดิม"
  • คลิกที่ปุ่มOKเพื่อบันทึกการตั้งค่า
Binary (Europe) Ltd

Safari 9.x+

  • ไปที่เมนู Safari และเลือกปรับแต่งค่า
  • คลิกที่แท็บความเป็นส่วนตัว
  • ตั้ง "บล็อกคุกกี้: " เป็นปฏิเสธ
  • คลิกที่ปุ่ม X ที่ด้านบนขวาเพื่อบันทึก และปิด
Binary (C.R.) S.A.

Opera 38+

  • ไปที่เมนูเครื่องมือ แล้วเลือกปรับแต่งค่า
  • คลิกที่แท็บขั้นสูง
  • เลือกคุกกี้จากคอลัมน์ด้านซ้าย
  • เลือกยอมรับคุกกี้
  • คลิกที่ปุ่มOKเพื่อบันทึกการตั้งค่าและปิด

  • ไปที่เมนูเครื่องมือ แล้วเลือกตัวเลือกอินเทอร์เน็ต
  • คลิกที่แท็บความเป็นส่วนตัว
  • คลิกที่ปุ่มขั้นสูง
  • เลือกการตั้งค่าคุกกี้อัตโนมัติ (ทับค่าเดิม)
  • เลือกยอมรับทั้งคุกกี้ของรายแรกและคุกกี้ของบุคคลที่สาม
  • คลิกที่ปุ่มOKเพื่อบันทึกการตั้งค่าและปิด
Binary Ltd

Microsoft Edge 12+

  • ไปที่เมนูดำเนินการเพิ่มเติม แล้วเลือกการตั้งค่า
  • ภายใต้การตั้งค่าขั้นสูง คลิกที่ดูการตั้งค่าขั้นสูง
